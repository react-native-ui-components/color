# @react-native-ui-components/color

Color module for React Native

## Installation

```sh
npm install @react-native-ui-components/color
```

## Usage

```js
import { colors } from '@react-native-ui-components/color';

// ...
<View style={{
	backgroundColor: colors.red
}}>
```
